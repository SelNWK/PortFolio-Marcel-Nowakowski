Stage : Création d'un site web pour l'association Ryoken Esport


Le but de ce stage était de créer un site web de présentation pour l'association Ryoken Esport, il a été effectué sur une durée de 7 semaines
( 4 janvier 2021 / 19 février 2021 )

J'ai donc effectué ce stage d'une durée de 7 semaines pour l'association Ryoken Esport , c'est une association Esportive qui a pour but d'encadrer des joueurs de jeux vidéo afin qu'il perfoment sur un jeu ( Encadrement coach analyste , coach mental , manager ...). Le but du stage était donc de crée un site web afin de présenter l'association ainsis que les membres de Ryoken Esport. J'ai effectué ce stage avec un camarade de classe en distanciel car Ryoken n'on pas de locaux .
